Protocols and specifications

This project uses Reliable UDP, which was created for the purposes of having stable TCP-like networking on top of UDP. The specification for it can be found here.

The protocol specification for the game's networking use itself (i.e. what's happening on top of the Reliable UDP layer) can be found here.

Code hierarchy

src/client/ contains all R-Type client code

src/client/components contains the code for all of the components of the ECS as used in the client

src/client/core contains the core code of the R-Type client, i.e. code necessary to all the other parts, such as much of the ECS's base code, code that wraps around window handling or code that handles networking

src/client/systems contains the code for all of the systems of the ECS as used in the client

src/client/utils contains code for a few miscellaneous things used all throughout the project but aren't some "core" part of it like the ECS

src/reliable-udp contains the code used for reliable networking on top of UDP, as specified in the Reliable UDP specification

src/server/ contains all R-Type server code

src/server/components contains the code for all of the components of the ECS as used in the server

src/server/core contains the core code of the R-Type server, i.e. code necessary to all the other parts, such as all of the core code around networking used by the rest of the server

src/server/packets contains the code for all of the packet types used with the R-Type networking, as specified in the R-Type Protocol specification

src/server/systems contains the code for all of the systems of the ECS as used in the server
